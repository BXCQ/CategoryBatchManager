# CategoryBatchManager是一款 Typecho 后台文章分类批量管理插件
主要功能：

- 在 **文章管理列表** 中提供批量操作：
  - **“移动到分类…”**：将选中文章的分类替换为所选分类；
  - **“添加分类…”**：在原有分类基础上追加新分类；
  - **“移除分类…”**：从文章已有分类中移除某些分类。
- 在文章列表每一行的 **“分类”列** 尾部增加一个小的 **“分类”** 链接：
  - 点击可快速为单篇文章调整分类。

整体操作基于 AJAX 完成，避免整页刷新，提升后台管理效率。
